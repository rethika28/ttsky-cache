## How it works
This project implements a simple direct-mapped cache memory using Verilog HDL for TinyTapeout ASIC flow.

The design supports:
- cache read operation
- cache write operation
- synchronous write
- asynchronous read

The cache contains:
- 4 memory locations
- 6-bit data storage
- 2-bit address input
